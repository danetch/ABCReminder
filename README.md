# World of Warcraft Addon

## Overview
This is a World of Warcraft addon designed to enhance your gameplay experience. It provides various features and utilities to assist players in their adventures.

## Installation
1. Download the addon files.
2. Extract the contents into your World of Warcraft `Interface/AddOns` directory.
3. Ensure the folder structure is maintained.

## Usage
- Load the game and navigate to the AddOns menu on the character selection screen.
- Enable the addon from the list.
- Once in-game, the addon will automatically initialize and start functioning.

## Features
- Core functionality implemented in `src/main.lua`.
- Utility functions available in `src/utils.lua`.
